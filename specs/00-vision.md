# NEURAX — Visión del Proyecto

## Concepto Central

NEURAX es un sistema personal de gamificación de vida real. Convierte cada hábito, actividad y decisión del usuario en progreso medible dentro de un ecosistema RPG de fantasía oscura. No es una app de productividad: es un **sistema de vida** donde el usuario construye su leyenda día a día.

## Nombre y Significado

**NEURAX** — fusión de "neuro" (mente, sistema nervioso, inteligencia) y el sufijo "x" (desconocido, poder, multiplicador). Representa la idea de un cerebro aumentado que conecta todas las áreas de la vida del usuario en un solo ecosistema inteligente.

## Tagline

> "Tu leyenda se construye cada día."
> "Convierte tu vida diaria en una aventura épica."

## Principios de Diseño del Producto

1. **Ecosistema único**: Todas las secciones están interconectadas. Una acción en Leonidas puede afectar Odin, que afecta el XP, que afecta el nivel.
2. **Fricción mínima**: Registrar una actividad debe tomar menos de 30 segundos en mobile.
3. **Recompensa inmediata**: Cada acción completada da retroalimentación visual instantánea (XP, animación, logro).
4. **Reglas claras**: El sistema tiene reglas estrictas donde las necesita y flexibilidad donde tiene sentido.
5. **Un solo usuario**: No hay comunidad, ranking, ni comparación. El único enemigo es la versión anterior del usuario.
6. **Inteligencia transversal**: Cronnos actúa como sistema nervioso central que conecta todas las secciones.

## Las 14 Secciones

| Nombre | Dominio |
|--------|---------|
| **Cronnos** | Calendario inteligente y gestión de tiempo |
| **Odin** | Misiones diarias, semanales y mensuales |
| **Demeter** | Finanzas, ingresos, egresos, patrimonio |
| **Soberbio** | Restaurantes y experiencias gastronómicas |
| **Dionisio** | Videos guardados de redes sociales |
| **Apolo** | Películas y calificación cinematográfica |
| **Alejandría** | Libros leídos y pendientes |
| **Michelin** | Recetas cocinadas y pendientes |
| **Odysseia** | Viajes y turismo |
| **Némesis** | Videojuegos |
| **Proeza** | Música y producción musical |
| **Leonidas** | Ejercicio físico |
| **Prodigy** | Estudios y academia |
| **Kubera** | Deseos materiales y metas de adquisición |

## Esencia que Siempre Debe Preservarse

- Sistema de XP con cálculo inteligente por tipo de actividad
- 6 Niveles con nombres RPG y progresión visual
- Sistema de Rachas con consecuencias reales al romperse
- Logros automáticos y manuales
- Estética RPG de fantasía oscura (paleta dorado/púrpura/negro)
- Dashboard móvil como pantalla principal
- Progreso personal sin comparación social

## Plataformas

| Plataforma | Rol | Prioridad |
|------------|-----|-----------|
| **App Móvil** (React Native) | Registro rápido de actividades, consulta de progreso, acceso a secciones simplificado | Principal |
| **Aplicación Web** | Gestión completa de todas las secciones, análisis profundo, configuración | Secundaria |

## Glosario

| Término | Definición |
|---------|------------|
| **XP** | Puntos de experiencia ganados por actividades y logros |
| **Racha** | Días consecutivos con al menos una actividad registrada |
| **Hito** | Logro manual especial creado por el usuario o sugerido por IA |
| **Misión** | Objetivo definido por el sistema en Odin con recompensa XP |
| **Área** | Clasificación de actividades: Rutinarias, Físicas, Económicas |
| **Sección** | Cada uno de los 14 módulos con nombre de la mitología |
| **Energía** | Indicador 0-100% del estado físico del usuario, afecta Cronnos |
